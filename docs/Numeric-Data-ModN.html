Type

data ModN n a where Source #

Newtype wrapper that represents \( \mathbb{Z}/n\mathbb{Z} \). ModN is a Ring i.e. supports addition, subtraction, and multiplication.

Optics

_MkModN :: (KnownNat n, Num a, Ord a, UpperBoundless a) => Lens' (ModN n a) a Source #

Lens' for ModN. Despite being a Lens, we use prism/iso syntax for consistency with other optics in this package and to witness that ModN is _nearly_ an Iso.

Examples

Expand
>>> import Optics.Core ((^.), (.~))
>>> n = mkModN @7 9
>>> n ^. _MkModN
2

>>> (_MkModN .~ 2) n
MkModN 2 (mod 7)
